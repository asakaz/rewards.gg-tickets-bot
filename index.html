Welcome to Rewards.GG Tickets Bot Official Download Page.

This bot is a very simple program that allows you to collect tickets on autopilot on Rewards.GG in a much more resource friendly and comfortable way.

Download

You can find the download at the top of the page, which was hosted on my GitHub Page.

Usage

The only file you need out of the archive that you download at the top of this page is the .exe (executable) file called: Rewards.gg Tickets Bot.exe.

First you log into your Rewards.GG account. Right now in the beta version there is support for social media platforms. You can only reach the steam, facebook, google, twitter, and twitch logins-- this is currently a bug so you have to use those.

Then next all you have to do is start the bot and minimize the program and check back later to check on your ticket count!
